# 登录
### 管理员登录
#### 请求数据
    /api/admin/login
#### 请求方式
    POST
#### 请求参数
    {
        "adminName" : "201806020000" ,// 工号：[字符串：必填]
        "adminPassword" : "123Aa_"  // 密码:[字符串：必填]
    }

{
    "ret":true,//是否登录成功 true成功 false失败
}
